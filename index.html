The video demonstrate how the Smartpos work. There are three types of users: admin, stock manager and cashier. The video illustrate the roles of each user in the system.



LOGIN

This is the window where users enter their respective login credentials and are authenticated to their rightful portals.




ADMIN PORTAL


DASH BOARD

This is the landing page for admin. It is mainly the statistical presentation of how the business perform in real time.




NAVIGATION BAR



INVENTORY

Here the admin see all the products available in the business.




admin is able to view each product details.




Admin can search the list of products.




Admin can also search the list of products by categories.




Admin view the list of sold products.




CHART

Here the admin view summary of behaviour of the business products using a pie chart.




MANAGE USERS (workers)

This is the window where the admin of the system manages the users of the system.
All the workers in the system are added and removed by the admin.
The admin is the super user can not be deleted or blocked.




Menu showing actions that can be done to each user.




Admin can view details for each user block the user and unblock the user.




Admin can change the profile picture for each user. Avatar is the default profile picture.




Admin can delete a user when is no longer employed in the business.




PAYROLL

This is the window where the admin of the system look the summary of how each user perform. The time that the user start working and stop working is shown. Also the initial stock the worker found, how much sold and how much damaged by the worker.







STOCK MANAGER PORTAL


DASH BOARD

This is the landing page for stock manager added by the admin. can see the statistical presentation of how the products perform in real time. Is able to see damaged products, sold products and products that need refill




Menu showing option of actions the manager can perform on single product.




Manager edit information of existing product.




Manager view existing product.




Manager upload picture for existing product.




Manager remove product from the system.




Manager is responsible for adding new products into the system.




DAMAGES

This is the window where the stock manager record damages.




List of added damages in the system including cashier's damages.




Refill Products

This is the window where the stock manager refill products that are about to finish. The limit to give a warning that a product need refill is determined at editing or adding a new product.




CASHIER PORTAL


SELL PRODUCTS

The cashier uses barcode scanner to read in products barcode to the barcode input. The system load the product to the list in the shopping cart. If the barcode is the same the system add the quantity of the same product but add product to the list otherwise. The total is shown automatically. The current product picture is shown and when the quantity input is changed it is added automatically to the current product in the cart.




List of products being added to the cart.




Searching the product to sell manually.




CASHIER DAMAGES

Damages directly connected to the cashier are recorded in this window.




RECEIPTS

Money worked by the cashier for the current day is shown in this window. Also all transactions total records in the firm are found here.
